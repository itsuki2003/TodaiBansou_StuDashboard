# 権限マトリクス

| 機能                       | SystemAdmin | Admin | Instructor |
| :------------------------- | :---------: | :---: | :--------: |
| **ユーザー管理**           |             |       |            |
| 講師の追加・編集・削除     |     ✅      |   -   |     -      |
| **生徒管理**               |             |       |            |
| 全生徒情報の閲覧           |     ✅      |  ✅   |     -      |
| 担当生徒情報の閲覧         |     ✅      |  ✅   |     ✅     |
| 生徒の新規登録（CSV）      |     ✅      |  ✅   |     -      |
| 担当生徒の編集             |     ✅      |  ✅   |     ✅     |
| 担当講師の割り当て         |     ✅      |  ✅   |     -      |
| **成績・スケジュール管理** |             |       |            |
| 担当生徒の成績登録・編集   |     ✅      |  ✅   |     ✅     |
| 担当生徒の予定登録・編集   |     ✅      |  ✅   |     ✅     |

_(✅: 可能, -: 不可)_
