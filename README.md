## 💻 使用技術スタック

### フロントエンド

- **フレームワーク: Next.js**
- **言語: TypeScript**
- **UI コンポーネント: Material-UI (MUI) or Chakra UI**
- **状態管理:** SWR (or React Query), Zustand
- **フォーム管理:** React Hook Form

### バックエンド

- **フレームワーク: NestJS**
- **言語: TypeScript**
- **データベース ORM: Prisma**

### データベース

- **PostgreSQL**

### インフラ

- **フロントエンドホスティング: Vercel**
- **バックエンドホスティング: AWS Fargate (ECS)**
- **データベース: AWS RDS for PostgreSQL**
- **ファイルストレージ: AWS S3**

### CI/CD

- **GitHub Actions**
