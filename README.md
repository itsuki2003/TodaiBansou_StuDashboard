# TodaiBansou_StuDashboard

オンライン家庭教師事業向けの生徒管理ダッシュボード

## 💻 使用技術スタック（予定）

- **フロントエンド:** Next.js (TypeScript)
- **バックエンド:** Ruby on Rails (API モード)
- **データベース:** PostgreSQL
- **インフラ:** AWS (EC2, RDS, S3)
- **CI/CD:** GitHub Actions

## 🚀 開発の始め方

1. このリポジトリをクローンします。
   ```bash
   git clone https://github.com/itsuki2003/TodaiBansou_StuDashboard.git
   ```
